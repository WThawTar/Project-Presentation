3. Import the database to PHPmyadmin
	Open from any type of browser 
	Type “localhost” on browser search
	And enter to PhpMyAdmin


	Once you have entered Click “New” for a new database on left hand side.
	And click import on top of the page.
	Once you are in import page click “choose file”
	Then you choose the file named “ispp.sql” and successfully import the file.
	Once you have imported you will now see your imported file on the left hand 	side bar of the page.
	If you see the imported file name ispp on left side bar “YOU HAVE SUCCESSFULLY 	IMPORTED THE DATABASE”.
